# Skills Box

一个收集和整理各类 AI 技能/智能体配置的资源库，帮助开发者快速配置和使用各种专业 AI 代理。

## 仓库结构

```
skills-box/
├── 个人/                    # 个人创建和整理的技能配置
│   └── 个人skill/
│       ├── codex配置提示词/  # Codex 相关配置和提示词
│       ├── 参考skills/       # 技能模板参考示例
│       └── 基于vue+ts+node/  # Vue+TS+Node 项目设计文档
│
└── 第三方/                   # 收集的第三方技能资源
    ├── agency-agents-main-CN/    # AI 代理机构中文化项目
    └── awesome-design-md-main/   # 设计相关 Markdown 资源
```

## 主要内容

### 个人技能

包含自定义的技能配置和参考模板：

- **Codex 配置提示词** - AI 编程助手的配置指南
- **参考 Skills** - 企业级项目技能模板，包括：
  - 企业级 Vue+TS+SCSS+Node+MongoDB 技能
  - 响应式企业技能
  - 电影级滚动效果技能
- **项目设计文档** - 基于 Vue+TypeScript+Node 的项目设计规范

### 第三方资源

#### Agency Agents (中文化)

一个完整的 AI 代理机构资源集合，涵盖多个专业领域：

| 部门 | 说明 |
|------|------|
| 工程部门 | 前端开发、后端架构、移动应用、AI工程、DevOps 等 |
| 设计部门 | UI设计、UX研究、品牌守护、视觉叙事等 |
| 营销部门 | 增长黑客、内容创作、SEO、社交媒体策略等 |
| 销售部门 | 外呼策略、交易策略、销售教练等 |
| 产品部门 | 产品经理、趋势研究、反馈综合等 |
| 项目管理 | 工作室运营、项目协调、实验追踪等 |
| 测试部门 | 证据收集、性能基准、API测试等 |
| 付费媒体 | PPC策略、搜索分析、程序化购买等 |
| 游戏开发 | Unity、Unreal、Godot、Blender 等 |
| 空间计算 | visionOS、XR、沉浸式开发等 |
| 专项领域 | 区块链安全、合规审计、法律服务等 |

#### Awesome Design MD

收集各知名公司的设计文档和规范，包括 Apple、Google、Stripe、Vercel、Notion 等公司的设计资源。

## 使用方法

1. 将所需的技能配置文件复制到你的项目根目录
2. 根据项目需求调整配置参数
3. 配合 AI 编程工具（如 Claude Code、Cursor 等）使用

## 适用场景

- 快速启动新项目，获得专业 AI 辅助
- 学习不同领域的最佳实践和规范
- 构建自定义的 AI 代理工作流
- 企业级项目的标准化开发流程

## 许可证

本仓库收集的资源分别遵循各自原始项目的许可证。
